# Exercise 01 - Create a Spring Web Project Using Maven

## Objective
Create a Spring Boot web application using Maven and verify successful application startup.

---

## Tools & Technologies
- Java 17
- Spring Boot 3.5.3
- Maven
- Spring Web
- Spring Boot DevTools
- VS Code

---

## Implementation
- Created a Spring Boot Maven project using VS Code Spring Initializr.
- Configured the project with the `com.cognizant` group and `spring-learn` artifact.
- Added Spring Web and Spring Boot DevTools dependencies.
- Included logging statements to verify execution of the `main()` method.
- Successfully built and ran the application.

---

## Output
The application started successfully on the embedded Tomcat server running on port 8080.

Refer to `output.png` for the execution output.

---

## Files in this Exercise

- **pom.xml**  
  Maven configuration file containing Spring Boot, Spring Web, and Spring Boot DevTools dependencies.

- **src/main/java/com/cognizant/springlearn/SpringLearnApplication.java**  
  Main Spring Boot application class containing the `main()` method and startup logging to verify successful application execution.

- **src/main/resources/application.properties**  
  Spring Boot configuration file used for application-level properties.

- **src/test/java/com/cognizant/springlearn/**  
  Package containing test classes generated by Spring Initializr.

- **output.png**  
  Screenshot showing successful build and startup of the Spring Boot application on the embedded Tomcat server.
