# Exercise 4: Creating and Configuring a Maven Project

## Scenario

You need to set up a new Maven project for the library management application and add the required Spring Framework dependencies.

---

## Steps

1. Create a new Maven project named **LibraryManagement**.
2. Add the required Spring dependencies to the `pom.xml` file.
3. Include dependencies for:
   - Spring Context
   - Spring AOP
   - Spring Web MVC
4. Configure the Maven Compiler Plugin to use Java 1.8.
5. Build the project using Maven to verify the configuration.

---

## Objective

The objective of this exercise is to understand how to create and configure a Maven project by managing dependencies and plugins required for a Spring Framework application.

---

## Solution Overview

In this exercise:

- A Maven project is created for the library management application.
- Spring Framework dependencies are added through the `pom.xml` file.
- The Maven Compiler Plugin is configured for Java 1.8.
- The project is built successfully using Maven to verify the project configuration.
