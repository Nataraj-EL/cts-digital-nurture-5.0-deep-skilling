# Exercise 01 – Git Configuration and Basic Commands

## Objective

This exercise demonstrates the basic Git workflow, including repository initialization, Git configuration, file tracking, committing changes, and verifying repository status.

## Tasks Performed

- Verified Git installation
- Checked global Git configuration
- Initialized a local Git repository
- Created a sample file (`welcome.txt`)
- Checked repository status
- Added the file to the staging area
- Committed the changes
- Verified the working tree status

## Technologies Used

- Git
- Git Bash / Terminal
- Ubuntu Linux

## Project Structure

```
GitDemo/
├── welcome.txt
└── README.md
```

## Learning Outcomes

- Understanding Git repository initialization
- Working with Git configuration
- Tracking files using Git
- Creating commits
- Checking repository status using Git commands
