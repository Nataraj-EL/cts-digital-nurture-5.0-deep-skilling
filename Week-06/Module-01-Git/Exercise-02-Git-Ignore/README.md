# Exercise 02 – Git Ignore

## Objective

This exercise demonstrates how to use the `.gitignore` file to exclude unwanted files and directories from version control.

## Tasks Performed

- Initialized a local Git repository
- Created a sample `.log` file
- Created a `log` directory
- Configured `.gitignore` to ignore log files and folders
- Verified ignored files using Git status

## Technologies Used

- Git
- Git Bash / Terminal
- Ubuntu Linux

## Project Structure

```
Exercise-02-Git-Ignore/
├── .gitignore
├── error.log
└── log/
    └── app.log
```

## Learning Outcomes

- Understanding the purpose of `.gitignore`
- Ignoring files using wildcard patterns
- Ignoring directories in Git
- Verifying ignored files with `git status`
