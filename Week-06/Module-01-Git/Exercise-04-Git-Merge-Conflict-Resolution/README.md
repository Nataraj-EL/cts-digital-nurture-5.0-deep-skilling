# Exercise 04 – Git Merge Conflict Resolution

## Objective

This exercise demonstrates how to create and resolve merge conflicts using Git branches.

## Tasks Performed

- Initialized a Git repository
- Created a new branch
- Added different versions of the same file
- Committed changes in both branches
- Compared branch differences
- Created a merge conflict
- Resolved the conflict manually
- Updated `.gitignore` to ignore backup files
- Deleted the merged branch
- Verified commit history

## Technologies Used

- Git
- Git Bash / Terminal
- Ubuntu Linux
