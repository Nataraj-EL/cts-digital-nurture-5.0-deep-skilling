# Exercise 03 – Git Branching and Merging

## Objective

This exercise demonstrates the use of Git branches for isolated development and merging changes back into the main branch.

## Tasks Performed

- Initialized a local Git repository
- Created a new Git branch
- Switched between branches
- Added and committed changes in the branch
- Compared branch differences
- Merged the branch into the master branch
- Viewed commit history using Git log
- Deleted the merged branch

## Technologies Used

- Git
- Git Bash / Terminal
- Ubuntu Linux


## Learning Outcomes

- Creating and switching Git branches
- Committing changes in different branches
- Comparing branches using Git
- Merging branches
- Viewing Git history
