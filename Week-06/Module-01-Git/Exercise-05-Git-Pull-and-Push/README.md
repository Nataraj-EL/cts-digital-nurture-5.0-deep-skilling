# Exercise 05 - Git Pull and Push

## Objective

Perform basic Git repository maintenance by verifying the repository state and executing pull and push operations with a remote repository.

## Tasks Performed

- Initialized a local Git repository
- Created and committed a README file
- Verified repository status using `git status`
- Listed available branches using `git branch`
- Executed the `git pull origin master` command
- Observed the terminal output for remote synchronization

## Technologies Used

- Git
- Git Bash / Terminal
- Ubuntu Linux
