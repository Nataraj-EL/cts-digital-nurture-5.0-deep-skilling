# Exercise 10: Implementing the MVC Pattern

## Scenario

You are developing a simple web application for managing student records using the MVC pattern.

---

## Steps

### 1. Create a New Java Project

Create a new Java project named **MVCPatternExample**.

---

### 2. Define Model Class

Create a class `Student` with attributes like:

- `name`
- `id`
- `grade`

---

### 3. Define View Class

Create a class `StudentView` with a method:

- `displayStudentDetails()`

---

### 4. Define Controller Class

Create a class `StudentController` that handles communication between the model and the view.

---

### 5. Test the MVC Implementation

Create a main class to demonstrate:

- Creating a Student object
- Updating student details using StudentController
- Displaying student details using StudentView

---

## Files in this Exercise

- **CONCEPT.md** - Explanation of MVC architecture and its applications.
- **src/Student.java** - Model class.
- **src/StudentView.java** - View class responsible for displaying details.
- **src/StudentController.java** - Controller class that connects Model and View.
- **src/MVCPatternTest.java** - Test class demonstrating MVC workflow.
- **output.png** - Screenshot of program output.
