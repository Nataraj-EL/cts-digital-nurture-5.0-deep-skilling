# SLF4J Exercise 1: Logging Error Messages and Warning Levels

## Scenario

You need to demonstrate logging error messages and warning messages using the SLF4J logging framework with Logback as the logging implementation.

---

## Steps

1. Add SLF4J and Logback dependencies to the Maven project.
2. Create a Java class that uses the SLF4J Logger.
3. Log an error message.
4. Log a warning message.
5. Run the application and observe the log output.

---

## Objective

The objective of this exercise is to understand how logging frameworks are used in Java applications to record important runtime information, warnings, and errors.

---

## Solution Overview

In this exercise:

- SLF4J is used as the logging API.
- Logback is used as the logging implementation.
- A simple Java application logs messages at different severity levels.
- Error and warning logs are displayed in the console.

---

## Files in this Exercise

- **pom.xml**  
  Maven configuration file containing SLF4J and Logback dependencies.

- **src/main/java/LoggingExample.java**  
  Java class demonstrating error and warning level logging using SLF4J.

- **output.png**  
  Screenshot showing successful execution and log messages.
