# Exercise 11 – React Event Handling

## Objective

This exercise demonstrates event handling in React using different types of event handlers and components.

## Features Implemented

- Counter component with Increment and Decrement buttons
- Multiple methods invoked on Increment button click
- Welcome message displayed using a button click event
- Synthetic event handling using the onClick event
- Currency Converter component to convert Indian Rupees (INR) to Euros (EUR)

## Technologies Used

- React
- JavaScript (ES6)
- JSX
- CSS
- Node.js
- npm
