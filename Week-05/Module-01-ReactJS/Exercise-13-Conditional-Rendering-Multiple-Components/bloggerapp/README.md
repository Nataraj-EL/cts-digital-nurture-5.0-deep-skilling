# Exercise 13 – Conditional Rendering with Multiple Components

## Objective

This exercise demonstrates different approaches to conditional rendering in React by displaying Book Details, Blog Details, and Course Details using reusable components.

## Features Implemented

- Book Details component
- Blog Details component
- Course Details component
- Conditional rendering using `if...else`
- Conditional rendering using the ternary operator (`? :`)
- Conditional rendering using the logical AND (`&&`) operator
- Rendering lists using the `map()` function
- Using unique keys while rendering list items

## Technologies Used

- React
- JavaScript (ES6)
- JSX
- CSS
- Node.js
- npm
