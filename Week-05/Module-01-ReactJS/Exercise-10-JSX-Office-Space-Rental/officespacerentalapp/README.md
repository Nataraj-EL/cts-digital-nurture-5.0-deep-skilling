# Exercise 10 – JSX Office Space Rental

## Objective

- Understand the fundamentals of JSX in React.
- Create and render React elements using JSX syntax.
- Use JavaScript expressions within JSX.
- Apply inline CSS styling dynamically.
- Render multiple objects using the `map()` function.

## Exercise Description

Develop a React application named **officespacerentalapp** that displays office space details using JSX.

The application should:

- Display a heading for the page.
- Display an image representing an office space.
- Create an office object containing details such as Name, Rent, and Address.
- Create a list of office objects and render them dynamically using JSX.
- Apply inline CSS to display the Rent value:
  - **Red** if the rent is below **60000**.
  - **Green** if the rent is **60000 or above**.

## Technologies Used

- React
- JSX
- JavaScript (ES6)
- CSS (Inline Styling)

## Output

The application successfully renders the office space details with dynamic rent color formatting based on the specified condition.
