# Exercise 12 – Conditional Rendering

## Objective

This exercise demonstrates conditional rendering in React by displaying different views for guest users and logged-in users.

## Features Implemented

- Guest view with flight details
- User view with ticket booking option
- Login and Logout functionality
- Conditional rendering using React state
- Flight details displayed in both views

## Technologies Used

- React
- JavaScript (ES6)
- JSX
- CSS
- Node.js
- npm

## Learning Outcomes

- Understanding conditional rendering in React
- Managing component state using Hooks

- Handling Login and Logout interactions
- Organizing applications using reusable components
